SECTION 501-09: Rear View Mirrors
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Exterior Mirror Cover

Removal

  1. Remove the exterior mirror glass. For additional information, refer to Exterior Mirror Glass In this section.
  1. If equipped, remove the exterior mirror cover retainer.
  1. Remove the exterior mirror cover.
    • Using a flat-blade screwdriver, release the 3 retaining tabs while pulling the exterior mirror cover toward the front of the vehicle.

Installation

  1. Align the inner lower corner of the mirror cover with the groove in the mirror frame.
  1. Align the 3 mirror cover tabs with the slots on the mirror frame and install the mirror cover.
    • Press the mirror cover into place and the 3 lock tabs are fully seated.
  1. If equipped, install the exterior mirror cover retainer.
  1. Install the exterior mirror glass. For additional information, refer to Exterior Mirror Glass In this section.
