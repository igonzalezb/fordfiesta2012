Procedure revision date: 07/15/2011

Window Glass Adjustment

  1. If needed, remove the door trim panel. For additional information, refer to Section 501-05 .
  1. If needed, remove the 2 access hole plugs.
  1. If needed, connect the window control switch, or install the window regulator handle.
  1. If needed, position the window glass in order to access the window glass clamp bolts.
  1. If needed, loosen the 2 window glass clamp bolts.

  1. NOTE: Make sure the window glass is correctly installed in door glass top run.

    Manually lift the window glass by hand, to the top of the door.

  1. NOTE: Make sure to lower the window glass evenly.

    NOTE: Make sure the window glass is seated in the bottom of the window glass clamps.

    Shift the glass rearward until it stops against the rear of the door glass top run while manually lowering the glass back down into the window glass clamps.
  1. Tighten the 2 window glass clamp bolts to 9 Nm (80 lb-in).
  1. Using the window control switch, fully raise the window glass and check that the window glass seats correctly in the upper door glass top run.
  1. If the window glass does not seat correctly, repeat steps 4 - 8.
  1. If this procedure was performed on the LH front window glass, the LH window motor must be initialized. For additional information, refer to Window Motor Initialization in this section.
