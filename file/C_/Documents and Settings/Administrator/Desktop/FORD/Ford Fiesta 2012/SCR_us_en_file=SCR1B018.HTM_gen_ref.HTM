SECTION 501-11: Glass, Frames and Mechanisms
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Window Glass — Rear Door

Removal

  1. Fully lower the rear door window glass.
  1. Remove the rear door trim panel. For additional information, refer to Section 501-05 .
  1. Remove the interior door window glass belt moulding.
  1. Remove and discard the exterior door window glass belt moulding.
  1. Position aside the lower rear portion of the rear door glass top run.
  1. Remove the 2 window extension screws and the window extension.
  1. Remove the 2 access hole plugs.
  1. Connect the window control switch, or install the window regulator handle.
  1. Position the window glass in order to access the window glass clamp bolts.
  1. Loosen the 2 window glass clamp bolts.
  1. Tilt the window glass forward and lift upward to remove the window glass from the door.

Installation

  1. Install the window glass into the door, but do not tighten the window glass clamps bolts.
  1. Position the window glass downward.
  1. Install the window extension and the tighten the 2 window extension screws.
  1. Install the lower rear portion of the rear door glass top run.

  1. NOTE: The window glass clamp bolts will be tightened when performing this step.

    Before tightening the window glass mounting bolts, make sure window glass is adjusted correctly. For additional information, refer to Window Glass Adjustment in this section.
  1. Install the 2 access hole plugs.
  1. Fully lower the rear door window glass.
  1. Install a new exterior door window glass belt moulding.
  1. Install the interior door window glass belt moulding.
  1. Install the rear door trim panel. For additional information, refer to Section 501-05 .
