SECTION 501-12: Instrument Panel and Console
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Power Point


Special Tool(s)
Remover, Power Point Socket
501-039 or equivalent

Removal

NOTE: Power point cover may differ depending on location.

  1. Open the power point cover.
  1. Install the Power Point Socket Remover in one of the power point socket slots.
  1. Position the Power Point Socket Remover so that it engages in the adjacent slot.
  1. Using the Power Point Socket Remover, pull the power point socket out of the retainer.
  1. Disconnect the electrical connector.

Installation

  1. Connect the electrical connector.
  1. Slide the power point socket into the retainer.
