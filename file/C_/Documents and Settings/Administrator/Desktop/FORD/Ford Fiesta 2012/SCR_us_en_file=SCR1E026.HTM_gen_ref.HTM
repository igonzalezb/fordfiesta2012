SECTION 501-14: Handles, Locks, Latches and Entry Systems
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Ignition Lock Cylinder


Item Part Number Description
1 11582 Ignition lock cylinder

Removal and Installation

  1. Remove the PATS transceiver. For additional information, refer to Section 419-01B .
  1. Remove the ignition lock cylinder.
    1. Position the ignition key to the ACC position.
    1. Inset a suitable tool into the ignition lock cylinder release access hole and depress the ignition lock cylinder release button.
    1. Pull outward on the ignition key and remove the ignition lock cylinder.
  1. To install, reverse the removal procedure.
