SECTION 501-16: Wipers and Washers
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Wiper Pivot Arm — Front

Removal

  1. Remove the front wiper pivot arm nut cover.
  1. Remove the front wiper pivot arm nut.
  1. Using a suitable 2 jaw puller, remove the front wiper pivot arm.

Installation

  1. Using a wire brush, clean the front wiper pivot arm shaft.

  1. NOTE: Make sure the wiper motor is in the full PARK position before installing the wiper pivot arms.

    Install the front wiper pivot arm so that the wiper blade tip is aligned to the "T" marks on the lower windshield glass.
  1. Install the front wiper pivot arm nut.
    • To install, tighten to 24 Nm (18 lb-ft).
  1. Install the front wiper pivot arm nut cover.
