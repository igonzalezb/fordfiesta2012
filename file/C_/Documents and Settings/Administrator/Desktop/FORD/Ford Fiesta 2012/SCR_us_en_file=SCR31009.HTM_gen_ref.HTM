Removal

  1. Remove the timing belt. For additional information, refer to Timing Belt in this section.
  1. Remove the crankshaft sprocket gear.
  1. Using the Input Shaft Oil Seal Remover and Slide Hammer, remove the crankshaft front oil seal.

Installation

  1. NOTE: Lubricate the oil seal with clean engine oil.

    Using the Crankshaft Vibration Damper Installer, Front Cover Oil Seal Installer and Front Crankshaft Seal Replacer, install the crankshaft front oil seal.
  1. Install the crankshaft sprocket gear.
  1. Install the timing belt. For additional information, refer to Timing Belt in this section.
