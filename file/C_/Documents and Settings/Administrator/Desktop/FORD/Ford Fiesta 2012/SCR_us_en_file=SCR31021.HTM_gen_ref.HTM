SECTION 303-01: Engine — 1.6L
2012 Fiesta Workshop Manual
IN-VEHICLE REPAIR
Procedure revision date: 07/15/2011

Valve Seals


Material
Item Specification
Motorcraft® SAE 5W-20 Premium Synthetic Blend Motor Oil (US); Motorcraft® SAE 5W-20 Super Premium Motor Oil (Canada)
XO-5W20-QSP (US); CXO-5W20-LSP12 (Canada) 		WSS-M2C945-A

Removal

NOTICE: During engine repair procedures, cleanliness is extremely important. Any foreign material, including any material created while cleaning gasket surfaces, that enters the oil passages, coolant passages or the oil pan can cause engine failure.

  1. Remove the valve springs. For additional information, refer to Valve Springs in this section.

  1. NOTE: Use valve stem seal pliers (such as Snap-on® YA-8230 or equivalent).

    Using valve stem seal pliers, remove and discard the valve seals.

Installation

  1. Install the valve stem seal installation sleeve.

  1. NOTE: Lubricate the valve stem seal with clean engine oil prior to installation.

    Using valve stem seal pliers, install the valve seals.
  1. Install the valve springs. For additional information, refer to Valve Springs in this section.
