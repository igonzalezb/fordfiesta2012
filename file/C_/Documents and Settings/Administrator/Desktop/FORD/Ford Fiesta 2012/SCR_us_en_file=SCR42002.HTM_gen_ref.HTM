Rear Suspension

The rear suspension consists of the following components:

  • Twist-beam axle (torsionally rigid axle)
  • Coil springs
  • Shock absorbers
  • Wheel bearing and wheel hub assemblies
  • Wheel studs

If compression takes place on both wheels evenly (for example when a load is placed on the vehicle), the whole axle unit swivels evenly in the rubber-metal bearings. If only one wheel is compressed, the cross-beam twists (undergoes torsion) and acts as a stabilizer. This minimizes track and toe changes.
