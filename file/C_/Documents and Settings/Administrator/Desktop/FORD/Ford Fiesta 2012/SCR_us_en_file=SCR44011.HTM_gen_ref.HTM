Tire Pressure Monitoring System (TPMS) Sensor

Disassembly

  1. NOTE: The Tire Pressure Monitoring System (TPMS) sensor cannot be removed without disassembly of the wheel and tire.

    Disassemble the wheel and tire. For additional information, refer to Wheel and Tire in this section.
