Differential Shim Selection


Special Tool(s)
Differential Bearing Preload Tool DPS6
307-681 (includes 307-681/1 and 307-681/2)
Shim Selection Set
308-164

NOTE: If the differential, differential bearings, clutch housing or transaxle case have been replaced this procedure must be performed.

  1. Install the differential.

  1. NOTE: Install the alignment standoffs at the transaxle dowel locations.

    NOTE: Place the special tools around the differential area as shown.

    Install the special tools.
  1. Position the differential bearing race on the differential. For removal of the differential bearing race from the clutch housing refer to Differential Shim in this section.
  1. Install the special tool on the differential.
  1. Position the clutch housing on the special tools and install the 6 special tool bolts, 4 shown.
    • Tighten to 18 Nm (13 lb-ft).
  1. Turn the differential several times clockwise and counter-clockwise to make sure the differential turns freely and that the bearing is seated.
  1. Turn the differential pre-load tool as shown.
  1. Loosen the differential pre-load tool bolt.

  1. NOTE: The shim size should be the average of the 3 measurements.

    Using a feeler gauge measure the clearance at points 1, 2 and 3 and record. Using the average of the recorded measurements, select the appropriate shim from the shim selection chart in the specifications section.
  1. Remove the special tools.
  1. Install the selected shim into the clutch housing and install the differential bearing race. For additional information, refer to Differential Shim in this section.
