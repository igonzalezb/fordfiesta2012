Removal

  1. Remove the front wheel and tire. For additional information, refer to Section 204-04 .
  1. Remove the drain plug and drain the transmission fluid.
  1. Install the drain plug.
    • Tighten to 43 Nm (32 lb-ft).
  1. Remove the brake hose retainer bolt and the brake hose retainer from the LH strut.
  1. Remove and discard the LH ball joint bolt and nut.
  1. Separate the LH ball joint from the knuckle.
  1. Using the special tools, remove the LH halfshaft assembly from the transaxle and position aside. Support the LH halfshaft assembly.
  1. Using a suitable tool, remove and discard the LH halfshaft seal.

Installation

  1. Install the LH halfshaft seal onto the special tool.
  1. Using the special tools, install the LH halfshaft seal.
  1. Using the LH halfshaft seal protector provided with the new LH halfshaft seal, install the LH halfshaft partially into the transaxle.
  1. Remove the LH halfshaft seal protector and install the LH halfshaft into the transaxle.
  1. Install the LH ball joint into the knuckle.
  1. Install the new LH ball joint bolt and nut.
    • Tighten to 48 Nm (35 lb-ft).
  1. Install the brake hose retainer and bolt on the LH strut.
    • Tighten to 25 Nm (18 lb-ft).
  1. Install the front wheel and tire. For additional information, refer to Section 204-04 .
  1. Fill the transaxle with transmission fluid. For additional information, refer to Transmission Draining and Filling in this section.
