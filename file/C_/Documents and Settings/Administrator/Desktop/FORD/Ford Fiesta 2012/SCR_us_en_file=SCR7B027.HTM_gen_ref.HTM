SECTION 307-11: Automatic Shifting Manual Transmission/Transaxle
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Inner Input Shaft Seal


Special Tool(s)
Remover, Outer Input Shaft Seal
307-673
Installer, Outer Input Seal
307-674 (includes 307-674/1 and 307-674/2)

Removal

  1. Remove the transaxle. For additional information, refer to Transaxle .
  1. Remove the clutch. For additional information, refer to Clutch .
  1. Using the special tools, remove and discard the inner input shaft seal.

Installation

  1. Using the special tools, install the new inner input shaft seal.
  1. Install the clutch. For additional information, refer to Clutch .
  1. Install the transaxle into the vehicle. For additional information, refer to Transaxle .
