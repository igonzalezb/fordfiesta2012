SECTION 307-11: Automatic Shifting Manual Transmission/Transaxle
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Input Shaft Speed Sensor — No. 1


Material
Item Specification
Motorcraft® Dual Clutch Transmission Fluid
XT-11-QDC 		WSS-M2C200-D2

Removal

  1. Disconnect the input shaft speed sensor 1 electrical connector.
  1. Remove the input shaft speed sensor 1.
    1. Remove the input shaft speed sensor 1 bolt.
    1. Remove the input shaft speed sensor 1.
    1. Remove the input shaft speed sensor 1 shim.

Installation

  1. Inspect the input shaft speed sensor 1 and transaxle bore for debris or damage. Clean, repair and/or replace as needed.

  1. NOTICE: Be sure to use the 3.5 mm (0.137 in) input shaft speed sensor 1 shim on input shaft speed sensor 1 or damage can occur to the transaxle.

    NOTE: Install a new O-ring if installing the existing input shaft speed sensor 1.

    NOTE: Use a small amount of transmission fluid on the input shaft speed sensor 1 O-ring for installation.

    Install the input shaft speed sensor 1.
    1. Install the input shaft speed sensor 1 shim.
    1. Install the input shaft speed sensor 1.
    1. Install the input shaft speed sensor 1 bolt.
      • Tighten to 10 Nm (89 lb-in).
  1. Connect the input shaft speed sensor 1 electrical connector.
