SECTION 307-11: Automatic Shifting Manual Transmission/Transaxle
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Manual Control Lever Shaft Seal


Special Tool(s)
Remover/Installer, Manual Lever Seal
307-670 (includes 307-670/1 and 307-670/2)

Removal

  1. Remove the transaxle. For additional information, refer to Transaxle in this section.
  1. Remove and discard the manual control lever bolt and remove the manual control lever.
  1. Install the special tool.

  1. NOTE: Use care to prevent clutch housing cup plug from being dislodged.

    Using the special tool and a suitable tool, apply slight pressure to the special tool, remove and discard the manual control lever seal.
  1. Inspect the transaxle case bore for debris or damage. Clean and/or repair as needed.

Installation

  1. Using the special tools, install the new manual control lever shaft seal.
  1. Install the manual control lever and the new manual control lever bolt.
    • Tighten to 26 Nm (19 lb-ft).
  1. Install the transaxle. For additional information, refer to Transaxle in this section.
