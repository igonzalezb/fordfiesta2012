SECTION 307-11: Automatic Shifting Manual Transmission/Transaxle
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Roll Restrictor

Removal

  1. With the vehicle in NEUTRAL, position it on a hoist. For additional information, refer to Section 100-02 .
  1. Remove the roll restrictor bolts and the roll restrictor.

Installation

  1. NOTE: The bolts are different lengths, make sure the bolts are in the correct location using the marks made during removal.

    Install the roll restrictor and the roll restrictor bolts.
    1. Tighten to 70 Nm (52 lb-ft).
    1. Tighten to 48 Nm (35 lb-ft).
