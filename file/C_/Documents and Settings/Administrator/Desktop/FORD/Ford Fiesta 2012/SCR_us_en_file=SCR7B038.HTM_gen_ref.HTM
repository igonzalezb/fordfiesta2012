Disassembly

  1. NOTE: Take note of the transmission wiring harness clip locations for installation.

    Remove the transmission wiring harness.
  1. Remove the 4 Transmission Control Module (TCM) bolts.
  1. Using a flat blade screwdriver, carefully pry the TCM from the transmission, at the location shown, and remove the TCM , discard the O-rings.
  1. Remove the 2 Transmission Range (TR) sensor bolts and the TR sensor.
  1. Remove the Output Shaft Speed (OSS) sensor bolt and OSS sensor.
  1. Remove the input shaft speed sensor 1.
    1. Remove the input shaft speed sensor 1 bolt.
    1. Remove the input shaft speed sensor 1.
    1. Remove the input shaft speed sensor 1 shim.
  1. Remove the input shaft speed sensor 2.
    1. Remove the input shaft speed sensor 2 bolt.
    1. Remove the input shaft speed sensor 2.
    1. Remove the input shaft speed sensor 2 shim.
  1. Install the special tools.
  1. Stand the transmission on its back.
  1. Remove the 4 clutch motor 1 bolts and remove clutch motor 1.
  1. Remove the 4 clutch motor 2 bolts and remove clutch motor 2.
  1. Remove and discard the clutch hub snap ring.

  1. NOTE: Index mark the clutch hub and clutch for installation.

    Remove the clutch hub.
  1. Install the special tools.
  1. Using the special tools, turn the center bolt of the special tool with a maximum of 5 Nm (44 lb-in) of torque to push the clutch down to release the pressure from the hollow input shaft snap ring.
  1. Remove and discard the hollow input shaft snap ring.

  1. NOTE: Make sure the lifting legs are installed on the 3 clutch studs marked with the manufacturing mark as shown.

    NOTE: Make sure the lifting leg with the angle is in the location shown.

    Install the special tools.
  1. Install the special tools.
  1. Turn the center bolt of the special tool with approximately 5 Nm (44 lb-in) of torque until the clutch is free of the input shaft.
  1. Using the special tools, remove the clutch as an assembly from the transmission.
  1. Using the special tool, remove and discard the inner input shaft seal.
  1. Remove the Z-washers from the engagement system bearing unit.

  1. NOTE: The engagement system bearing unit should be removed easily.

    Remove the engagement system bearing unit. Inspect for wear or damage. Replace as necessary.

  1. NOTICE: Remove the spring retainer bolts by loosening them slightly in an alternating pattern or damage can occur to the clutch actuator levers.

    NOTE: The spring retainers and actuator levers are matched sets with corresponding serial numbers. If the spring retainers and actuator levers are being reused, mark on each spring retainer and actuator lever for assembly.

    Remove the 4 spring retainer bolts from the spring retainers and remove the spring retainers.
  1. Install clutch actuator lever locks as shown if available.
  1. Remove the 2 bolts from the clutch actuator lever 1.
  1. Remove the 2 bolts from the clutch actuator lever 2.

  1. NOTICE: Be sure to properly handle the actuator lever as shown or damage may occur.

    Remove the clutch actuator lever 1.

  1. NOTICE: Be sure to properly handle the actuator lever as shown or damage may occur.

    Remove the clutch actuator lever 2.
  1. Remove the guide tube bolts.
  1. Remove the guide tube.
  1. Using the special tools, remove and discard the input shaft case seal.
  1. Remove and discard the input shaft snap ring.
  1. Remove discard the manual control lever bolt and remove the manual control lever.
  1. Remove the top 3 clutch housing-to-case bolts.
  1. Remove the bottom 18 clutch housing-to-case bolts (13 shown).
  1. Install the special tool.

  1. NOTE: Install the special tools in the position shown.

    Install the special tools and special tools bolts.
    • Tighten to 18 Nm (159 lb-in).
  1. Turn the special tool bolt approximately 20 turns to separate the clutch housing from the transmission case.
  1. Lift the clutch housing off the transmission case.
  1. Remove and discard the clutch housing input shaft bearing retainer bolts and remove clutch housing input shaft bearing retainer.
  1. By lightly tapping, remove and discard the clutch housing input shaft bearing.
  1. Remove the differential.
  1. Remove the park pawl actuation shaft assembly.
  1. Remove the park pawl rod and park pawl.
  1. Remove the large clock gears and clock gear pins.
  1. Remove the small clock gears and clock gear pins.
  1. Remove the magnet.
  1. Install the special tool on the transmission case and gear set assembly.
  1. Turn the transmission case and gear set assembly over on the special tool.
  1. Remove the special tool from the transmission case.
  1. Remove and discard the 6 output shaft bearing retainer bolts.
  1. Lift the transmission case off the gearset assemblies.
  1. Remove the park plunger plug.
  1. Remove the park plunger.
  1. Remove the oil spout.
  1. Remove the 4 shift fork rods.
  1. Remove the 2 shift drums.
  1. Remove the shift forks and linkages.
  1. Install the special tool.
  1. Turn the gearset assembly over on its side.
  1. Remove the special tool.
  1. Remove output shaft 1 assembly by removing the special tool.
  1. Remove the input shaft assembly by removing the special tool.
  1. Remove the special tool from output shaft 2 assembly.
  1. Remove the hollow input shaft from the solid input shaft.

Assembly

  1. Install the hollow input shaft onto the inner input shaft.

  1. NOTE: If the clearance is less than the minimum measurement stated in the specifications section, install a new shift fork.

    Using a suitable feeler gauge, measure the clearance between the shift fork and synchronizer hub sleeve in various places around the circumference.

  1. NOTE: Apply pressure on the synchronizer ring while taking the measurements.

    NOTE: If the clearance is less than the minimum measurement stated in the specifications section, install a new output shaft assembly.

    Using a suitable feeler gauge, measure the clearance between the synchronizer ring and gear in various places around the circumference.

  1. NOTE: Make sure the synchronizers are in the neutral position.

    Install output shaft 2 assembly on the special tool.
  1. Install the input shaft assembly on the special tool.

  1. NOTE: Make sure the synchronizers are in the neutral position.

    Install output shaft 1 assembly on the special tool.
  1. Install the special tool.
  1. Turn the gearset assembly over onto the special tool.
  1. Remove the special tool.
  1. Install the 4th/reverse gear shift fork.
  1. Install the 2nd/6th gear shift fork and linkage.

  1. NOTE: Make sure shift drum 1 is in the neutral position as shown.

    Install shift drum 1.

  1. NOTE: Make sure shift drum 2 is in the neutral position as shown.

    Install the 3rd gear linkage and shift drum 2.
  1. Install the 1st/5th gear shift fork.
  1. Install the 3rd gear shift fork.
  1. Install the 4 shift fork rods.
    1. Install the small shift fork rods.
    1. Install the large shift fork rods.
  1. Install the oil spout.

  1. NOTE: Make sure the output shaft bearing retainer brackets are positioned as shown.

    Install the special tools on the output shafts assemblies.
  1. Install the transmission case onto the gearset assemblies.
  1. Install the 6 new output shaft bearing retainer bolts. Tighten in 2 stages.
    • Stage 1: tighten to 10 Nm (89 lb-in).
    • Stage 2: tighten to 32 Nm (24 lb-ft).
  1. Install the special tool on the transmission case.
  1. Turn the transmissioncase and gearset assembly over onto the special tool.
  1. Remove the special tool.
  1. Install the magnet.
  1. Install the small clock gears and clock gear pins.
  1. Install the large clock gears and clock gear pins.
  1. Install the park pawl and park pawl rod.

  1. NOTE: Make sure the pin is installed as shown.

    Install the park pawl actuation shaft assembly.

  1. NOTE: If the differential bearings, differential, clutch housing and/or transmission case are replaced refer to Differential Shim Selection in this section.

    Install the differential.

  1. NOTICE: Make sure the transmission case and the clutch housing are clean and no old sealant is present or transmission fluid leaks can occur and cause transmission damage.

    NOTICE: Make sure the park plunger and park plunger plug were removed from the the clutch housing or damage to the park pawl actuation shaft assembly (rooster comb) will occur.

    Apply a 1 mm (0.039 in) bead of Gasket Maker around the inside of the bolt holes on the case side and completely around each dowel.
  1. Using the special tools, install the new clutch housing input shaft bearing.
  1. Install the clutch housing input shaft bearing retainer and the new clutch housing input shaft bearing retainer bolts.
    • Tighten to 11 Nm (97 lb-in).

  1. NOTICE: Do not overheat the bearing or damage to the bearing may occur.

    Using a 1500W heat gun, or equivalent, heat the input shaft bearing in the clutch housing for approximately 5 minutes.
  1. Install the clutch housing onto the transmission case.
  1. Using the special tool and a hammer, tap the clutch housing down onto the transmission case.
  1. Install the 18 bottom clutch housing-to-case bolts (13 shown).
    • Tighten to 27 Nm (20 lb-ft).
  1. Install the 3 top clutch housing-to-case bolts.
    • Tighten to 27 Nm (20 lb-ft).
  1. Install the park plunger.
  1. Install the park plunger plug.
    • Tighten to 43 Nm (32 lb-ft).
  1. Install the manual control lever and the new manual control lever bolt.
    • Tighten to 26 Nm (19 lb-ft).
  1. Install a new input shaft snap ring.
  1. Using the special tools, install a new input shaft case seal.
  1. Using the locating flange on the guide tube, install the guide tube.

  1. NOTE: The guide tube bolts must not project beyond the mounting surface of the clutch engagement levers.

    Install the 3 guide tube bolts.
    • Tighten to 8 Nm (71 lb-in).
  1. Inspect the guide tube for wear or damage. Replace as necessary.

  1. NOTE: One clutch actuator lever bushing shown, one per clutch actuator lever.

    Install the clutch actuator lever bushings.

  1. NOTICE: Be sure to properly handle the actuator lever as shown or damage may occur.

    Install the clutch actuator lever 2.
  1. Install clutch motor 2 without the bolts to aide in the alignment of the clutch actuator lever.
  1. Install the 2 clutch actuator lever 2 bolts.
    • Tighten to 19 Nm (168 lb-in).

  1. NOTICE: Be sure to properly handle the actuator lever as shown or damage may occur.

    Install the clutch actuator lever 1.
  1. Install clutch motor 1 without the bolts to aide in the alignment of the clutch actuator lever.
  1. Install the 2 clutch actuator lever 1 bolts.
    • Tighten to 19 Nm (168 lb-in).

  1. NOTICE: Tighten the spring retainer bolts by tightening them slightly in an alternating pattern or damage can occur to the clutch actuator levers.

    NOTE: Make sure the spring retainer contour aligns with the actuator lever contour as shown.

    NOTE: Make sure the serial number on the spring retainer matches the serial number on the actuator lever. If reusing the spring retainers and actuator levers use the marks made during removal for assembly.

    Install the spring retainers and spring retainer bolts.
    • Tighten to 26 Nm (19 lb-ft).
  1. Remove the clutch actuator lever locks if installed.

  1. NOTICE: Do not apply grease or other lubricants to the clutch actuator lever fingers, the engagement system bearing, or the Z-washers or clutch contamination can occur.

    NOTE: The engagement system bearing unit should be installed easily.

    Install the engagement system bearing unit.

  1. NOTICE: Do not tilt transmission after the Z-washers are installed, they may come off center and damage can occur.

    Install the Z-washers onto the engagement system bearing unit and center them correctly.
  1. If installing a new clutch go to step 70.

  1. NOTE: Make sure the special tool washer is also installed.

    Install the special tool in a vise.
  1. Place the clutch on the special tool and align one clutch stud with the slotted hole on the special tool.
  1. Install the special tool loosely on the clutch.
  1. Install the special tool on the clutch making sure the legs are inserted in the slots.
  1. Turn the special tool clockwise to the stops and hold.

  1. NOTICE: Failure to rotate the self-adjuster to the stops before compressing the clutch, causes damage to the clutch.

    Tighten the special tool until the 3 clutch 1 locks can be locked in place by hand.
  1. Install the special tool loosely on the clutch.
  1. Using the special tool, move the clutch self-adjuster counterclockwise to the stop and hold.

  1. NOTICE: Make sure the clutch self-adjuster is held to the stops before compressing the clutch or damage will occur to the clutch.

    NOTE: An audible click can be heard when the clutch locks lock.

    While holding the clutch self-adjuster to the stops, use the special tool and compress the clutch to the stops on the clutch. The locks on clutch 2 lock at this time.
  1. Remove the special tools.

  1. NOTE: The minimum number of engaged clutch locks is 6. Make sure the 6 locked clutch fingers align with the clutch studs as shown. If not, reset the clutch 2 locks as necessary.

    Visually check the locks to make sure they have locked. 4 locks shown, 12 total.
  1. If installing a used clutch, verify the clutch is not worn or damaged. Perform the next 2 steps to check for clutch wear.
  1. Check clutch 1 for wear by measuring the distance between the adjuster ring 1 long hole and lever 1 spring. Replace the clutch assembly if the measurement is 26 mm (1.023 in) or less.
  1. Check clutch 2 for wear by measuring the distance between adjuster ring 2 and the cover. Replace the clutch assembly if the measurement is 15 mm (0.59 in) or less.
  1. Using the special tools, install the new inner input shaft seal.
  1. Apply a small amount of transmission fluid on the input shaft bearing surface.
  1. Apply a small amount of DPS6 Grease on both input shaft splines.
  1. Verify the Z-washers are installed correctly.

  1. NOTICE: Verify the clutch locks are locked prior to installation or the transmission may not operate correctly and damage can occur. If the clutch locks are not locked go to step 56 within this procedure.

    NOTE: Make sure the angled lifting leg is installed in the location shown.

    Using the special tools, install the clutch in the transmission.
  1. Remove all the special tools.
  1. Install the special tools onto the transmission case.

  1. NOTICE: Do not overtighten the special tool or damage can occur.

    Using the special tools, push the clutch down with approximately 5 Nm (44 lb-in) of torque until seated on the input shaft bearing and the snap ring groove is visible.

  1. NOTE: Make sure the hollow input shaft snap ring is fully seated.

    Install the new hollow input shaft snap ring.

  1. NOTE: Align the index marks on the clutch hub and clutch.

    Install the clutch hub.
  1. Install the new clutch hub snap ring in the position shown.

  1. NOTICE: Do not turn the special tool more than 12 turns or damage to the clutch actuator engagement lever can occur.

    Using the special tool, release the clutch 1 locks by slowly turning the special tool counterclockwise. An audible click will be heard when the clutch 1 lock is released. Repeat step to release clutch 2 locks.
  1. Apply High Temperature Nickel Anti-Seize Lubricant to both clutch motor surfaces where indicated.
  1. Apply High Temperature Nickel Anti-Seize Lubricant to the clutch actuator lever splines as shown.
  1. Install the clutch motor 2 and the 4 clutch motor 2 bolts.
    • Tighten to 5 Nm (44 lb-in).
  1. Install the clutch motor 1 and the 4 clutch motor 1 bolts.
    • Tighten to 5 Nm (44 lb-in).

  1. NOTICE: Be sure to use the 3.2 mm (0.125 in) input shaft speed sensor 2 shim or damage can occur to the transmission.

    NOTE: Install a new O-ring if installing the existing input shaft speed sensor 2.

    NOTE: Use a small amount of transmission fluid on the input shaft speed sensor 2 O-ring for installation.

    Install the input shaft speed sensor 2.
    1. Install the input shaft speed sensor 2 shim.
    1. Install the input shaft speed sensor 2.
    1. Install the input shaft speed sensor 2 bolt.
      • Tighten to 10 Nm (89 lb-in).

  1. NOTICE: Be sure to use the 3.5 mm (0.137 in) input shaft speed sensor 1 shim on input shaft speed sensor 1 or damage can occur to the transmission.

    NOTE: Install a new O-ring if installing the existing input shaft speed sensor 1.

    NOTE: Use a small amount of transmission fluid on the input shaft speed sensor 1 O-ring for installation.

    Install the input shaft speed sensor 1.
    1. Install the input shaft speed sensor 1 shim.
    1. Install the input shaft speed sensor 1.
    1. Install the input shaft speed sensor 1 bolt.
      • Tighten to 10 Nm (89 lb-in).

  1. NOTE: If installing the old OSS sensor, install a new O-ring.

    Install the OSS sensor and the OSS sensor bolt.
    • Tighten to 10 Nm (89 lb-in).
  1. Install the TR sensor and the 2 TR sensor bolts.
    • Tighten to 10 Nm (89 lb-in).
  1. Carefully inspect the TCM and the transmission bore for debris or damage. Clean and/or repair as needed.

  1. NOTE: Use a small amount of transmission fluid on the new O-rings prior to installation.

    Install new O-rings if the TCM is being reused.

  1. NOTICE: Do not force the TCM onto the transmission or damage may occur.

    NOTICE: Make sure the TCM sits flush on the transmission, or damage can occur.

    Install the TCM . Apply light pressure to seat the O-rings. If the TCM does not push in place with light pressure, remove the TCM and slightly turn the TCM gears and retry installing the TCM .
  1. Install the 4 TCM bolts.
    • Tighten to 27 Nm (20 lb-ft).
  1. Install the wiring harness.
