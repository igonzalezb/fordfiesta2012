Disassembly

  1. Using the special tools, remove the clutch housing differential bearing race and shim.

Assembly

  1. Install the selected differential bearing shim. If the differential bearings, differential, clutch housing or transaxle case were replaced, refer to Differential Shim Selection in this section.
  1. Using the special tools, install the differential bearing race.
