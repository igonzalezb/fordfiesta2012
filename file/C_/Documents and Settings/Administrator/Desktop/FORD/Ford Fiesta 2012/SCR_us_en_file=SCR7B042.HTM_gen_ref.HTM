2012 Fiesta Workshop Manual
DISASSEMBLY AND ASSEMBLY OF SUBASSEMBLIES
Procedure revision date: 07/15/2011

Input Shaft — Inner


Special Tool(s)
Puller, Bearing
205-D064


Item Part Number Description
1 7717 Inner input shaft
2 7B412 Radial bearing
3 7B412 Roller bearing
4 7M150 Needle bearing
5 7M150 Needle bearing

Disassembly

  1. Remove the inner input shaft needle bearing.
  1. Remove the inner input shaft needle bearing.
  1. Remove the inner input shaft roller bearing.
  1. Using the special tools and a press, remove the inner input shaft radial bearing.

Assembly

  1. Inspect the inner input shaft for wear or damage, replace as needed.
  1. Using the special tools and a press, install the inner input shaft radial bearing.
  1. Install the inner input shaft roller bearing.
  1. Install the inner input shaft needle bearing.
  1. Install the inner input shaft needle bearing.
