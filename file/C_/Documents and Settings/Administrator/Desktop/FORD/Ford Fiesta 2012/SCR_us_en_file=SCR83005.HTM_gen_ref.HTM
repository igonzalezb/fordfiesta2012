Removal

  1. Remove the LH halfshaft. For additional information, refer to Section 205-04 .
  1. Using the Pilot Bearing Remover with the Halfshaft Oil Seal Remover, remove and discard the halfshaft oil seal.

Installation

  1. Using the Halfshaft Oil Seal Installer, install the new halfshaft oil seal.
  1. Install the halfshaft. For additional information, refer to Section 205-04 .

  1. NOTE: The fill height is 0.0-0.5 mm (0.0-0.2 in) below the lower edge of the filler hole.

    Remove the transaxle fluid filler plug. Check the transaxle fluid level and fill with manual transmission fluid as necessary.
    • Tighten to 35 Nm (26 lb-ft).
