Disassembly

  1. Remove the bearing.
    • Remove and discard the circlip.
    • Using the special tools, remove and discard the bearing.
  1. Remove 4th gear, remove and discard the snap-ring and remove the 3rd/4th gear synchronizer.
  1. Remove the 3rd gear synchronizer ring and 3rd gear.
  1. Remove the 2 half moon washers and 2nd gear.
  1. Remove the inner and outer synchronizer rings and the synchronizer cone.
  1. Remove and discard the snap-ring and remove the 1st/2nd gear synchronizer with reverse gear.
  1. Remove the outer synchronizer ring.
  1. Remove the synchronizer cone, inner synchronizer ring and the 1st gear.
  1. Remove the bearing cone.
    • Remove and discard the snap-ring.
    • Using the special tools, remove and discard the bearing cone.

Assembly

  1. Install the new bearing cone and new snap-ring.
    1. Evenly heat the new bearing cone to approximately 176°F (80°C) and slide it onto the output shaft.
    1. Install the new snap-ring.
  1. Install the 1st gear, inner synchronizer ring and the synchronizer cone.
  1. Install the outer synchronizer ring.
  1. Install the new snap-ring.
  1. Install the synchronizer cone and the outer and inner synchronizer rings.
  1. Install the 2nd gear and the 2 half moon washers.
  1. Install the 3rd gear and 3rd gear synchronizer ring.
  1. Install the 3rd/4th gear synchronizer and new snap-ring.
  1. Install 4th gear and using a suitable tube, install the new bearing and new circlip.
