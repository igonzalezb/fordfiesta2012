REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Heater Core And Evaporator Core Housing


Item Part Number Description
1 19D888 Audio unit vent tube
2 W505422 Heater core and evaporator core housing bolt
3 W700430 Heater core and evaporator core housing nuts (4 required)
4 19850 Heater core and evaporator core housing

Removal and Installation

  1. Remove the instrument panel. For additional information, refer to Section 501-12 .
  1. Remove the 2 floor ducts.
  1. Disconnect the electrical connectors and detach the wire harness clips from the RH side of the heater core and evaporator core housing.
  1. Disconnect the electrical connectors and detach the wire harness retainer from the LH side of the heater core and evaporator core housing.
  1. Release the clamp and detach the audio unit vent tube from the heater core and evaporator core housing.
  1. Remove the 4 heater core and evaporator core housing nuts, and the heater core and evaporator core housing bolt.
    • To install, tighten to 6 Nm (53 lb-in).
  1. To install, reverse the removal procedure.
