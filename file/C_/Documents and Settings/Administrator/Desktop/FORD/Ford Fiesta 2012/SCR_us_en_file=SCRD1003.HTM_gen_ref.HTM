Warning Chimes

The warning chimes provide the driver with an audible warning that acts as a supplemental alert to the Instrument Panel Cluster (IPC) indication and message center warnings. The IPC controls all warning chimes based on messages received from external modules.

The warning chime system consists of the following:

  • IPC
  • Safety belt warning indicator switch(es)
  • Door ajar switch(es)
  • Headlamp switch
  • Key-in-ignition warning switch (without Intelligent Access (IA))
  • Parking brake warning indicator switch
  • Body Control Module (BCM)
