Key Programming — Program a Key Using Two Programmed Keys

NOTE: This procedure is only for vehicles without Intelligent Access (IA). For vehicles with IA the keys can only be programmed using a scan tool.

NOTE: This procedure works only if 2 or more programmed keys are available. If 2 programmed keys are not available, refer to Key Programming Using Diagnostic Equipment in this section.

NOTE: This procedure only programs the Passive Anti-Theft System (PATS) portion of the key into the Instrument Panel Cluster (IPC). It does not program the Remote Keyless Entry (RKE) Transmitter Identification Code (TIC) into the Body Control Module (BCM).

  1. Insert the first programmed PATS key into the ignition and turn the key from the OFF position to the RUN position (maintain the key in the RUN position for approximately 3 seconds).
  1. Turn the first key to the OFF position and remove the key from the ignition.
  1. Within 10 seconds of turning the key to the OFF position, insert a second programmed PATS key into the ignition and turn the key from the OFF position to the RUN position (maintain the key in the RUN position for approximately 3 seconds).
  1. Turn the second key to the OFF position and remove the key from the ignition.
  1. Within 10 seconds of turning the second key to the OFF position, insert the new, unprogrammed PATS key into the ignition and turn the key from the OFF position to the RUN position (maintain the key in the RUN position for approximately 3 seconds).
  1. If it is desired to program additional key(s) (up to a total of 8 keys can be programmed), repeat Steps 1 - 5 for each additional key.
  1. Refer to RKE Transmitter Programming in Section 501-14 to program the RKE portion of the key.
