SECTION 419-01B: Anti-Theft — Passive Anti-Theft System (PATS)
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Passive Start Antenna — Front


Item Part Number Description
1 16K262 Scrivet (2 required)
2 — Front passive start antenna electrical connector (part of 14K024)
3 W505253 Front passive start antenna bracket bolt (2 required)
4 15603 Front passive start antenna
5 19H217 Front passive start antenna bracket

Removal and Installation

  1. Remove the floor console finish panel. For additional information, refer to the exploded view In Section 501-12 .
    • Lift upward on the rear of the floor console finish panel to release the floor console finish panel clips.
    • If equipped, disconnect the electrical connector.
    • Slide the park brake lever boot and the floor console finish panel over the park brake lever.
  1. Pull the red lock tab outward and disconnect the front passive start antenna electrical connector.
  1. Remove the 2 bolts and the front passive start antenna and bracket assembly.
    • To install, tighten to 13 Nm (10 lb-ft).
  1. If required, remove the 2 scrivets and the front passive start antenna.
  1. To install, reverse the removal procedure.
