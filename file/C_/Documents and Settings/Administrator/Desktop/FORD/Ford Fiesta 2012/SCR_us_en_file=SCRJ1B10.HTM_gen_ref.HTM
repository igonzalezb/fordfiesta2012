SECTION 419-01B: Anti-Theft — Passive Anti-Theft System (PATS)
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Passive Start Antenna — Center


Item Part Number Description
1 — Center passive start antenna electrical connector (part of 14A005)
2 15603 Center passive start antenna
3 W713358 Center passive start antenna bracket (part of 15603)

Removal and Installation

  1. Remove the floor console. For additional information, refer to Section 501-12 .
  1. Pull the red lock tab outward and disconnect the rear floor console passive start antenna electrical connector.
  1. Unclip and remove the center passive start antenna and bracket assembly.
  1. If required, remove the center passive start antenna from the center passive start antenna bracket.
  1. To install, reverse the removal procedure.
