SECTION 419-10: Multifunction Electronic Modules
2012 Fiesta Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 07/15/2011

Remote Function Actuator (RFA) Module


Special Tool(s)
Vehicle Communication Module (VCM) and Integrated Diagnostic System (IDS) software with appropriate hardware, or equivalent scan tool


Item Part Number Description
1 W503922 Remote Function Actuator (RFA) bracket bolt (2 required)
2 — RFA bracket
3 — RFA module J-nut (2 required)
4 19G481 RFA module
5 — RFA module electrical connectors (part of 14A005)
6 — RFA module screw (2 required)

Removal and Installation

  1. NOTE: The IA keys need to be programmed into the new RFA module during ignition off. For additional information, refer to Key Programming Using Diagnostic Equipment in Section 419-01B . After the IA keys have been programmed into the new RFA module, the ignition must be turned to the ON mode in order to reset the parameters in the new RFA module and the PCM. For additional information, refer to Passive Anti-Theft System (PATS) Parameter Reset in Section 419-01B .

    Remove the passenger seat. For additional information, refer to Section 501-10 .
  1. Disconnect the 5 RFA module electrical connectors.
  1. Remove the 2 bolts and the RFA module and bracket assembly.
    • To install, tighten to 10 Nm (89 lb-in).
  1. To install, reverse the removal procedure.
    • Program the IA keys into the new RFA module. For additional information, refer to Key Programming Using Diagnostic Equipment in Section 419-01B .
    • Carry out a Passive Anti-Theft System (PATS) parameter reset. For additional information, refer to Passive Anti-Theft System (PATS) Parameter Reset in Section 419-01B .
